Force-Field Particle Animator

This is a generative art / animation project. The project is open source, so feel free to use it for whatever you wish.

There are a few hotkeys which can speed up using the tool:

  • Press "r" to reset the canvas. This will move all particles back to their original position in the source image
  • Press "s" to save a screenshot of the current state of the animation (png image)
  • Press "v" to start recording a video of the animation. Press v again to stop the recording. An mp4 video file will be exported to your downloads folder afterwards
